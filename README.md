# Marketing & Campaign Performance Analysis (2023 - 2025)

An end-to-end data analytics project focused on analyzing marketing campaigns, customer engagement, and product performance over a three-year period (2023 - 2025). The workflow involved data ingestion, comprehensive data cleaning and view creation using **SQL**, followed by building a dynamic, multi-page interactive dashboard in **Microsoft Excel**.

## 💡 Key Metrics & Insights
* **Total Engagement:** 14.6M interactions (consisting of 11.8M Views, 2.3M Clicks, and 0.53M Likes).
* **Conversion Metrics:** Overall conversion rate stands at **5.30%**, with a **49.87%** potential conversion rate.
* **Customer Feedback:** 1,363 Total Reviews (844 Satisfied vs. 519 Dissatisfied).
* **Funnel & Content Trends:** Analyzes user actions (Click, Drop-off, View) across stages (Homepage, Product page, Checkout) and content types (Video, Social Media, Newsletter, Blog).
* **Product & Regional Analysis:** 
  * Segments products by campaign performance and satisfaction rate (e.g., Climbing Rope, Ski Boots).
  * Evaluates country-specific actions and conversion rates across global markets (Spain, Italy, Germany, UK, etc.).

## 🛠️ Data Pipeline & Technologies Used
1. **Data Cleaning & Transformation (SQL):** 
   * Cleaned raw marketing data, handled missing values, and filtered anomalies.
   * Created optimized **SQL Views** to structure the data efficiently for reporting.
2. **Data Visualization (Microsoft Excel):**
   * Imported SQL Views into Excel.
   * Built advanced Pivot Tables and interactive charts.
   * Integrated global **Slicers** for dynamic filtering by *Years, Actions, Stages,* and *Review Categories*.

## 📁 Project Structure
* `Views SQL Queries/` - Folder containing SQL scripts for data cleaning and view creation.
* `Cleaned Data SET/` - The interactive Excel workbook.
* `Dataset/` - Sql Table Raw Data.

## 🚀 How to Explore the Project
1. Navigate to the `Views SQL Queries/` folder to review the data cleaning pipeline.
2. or Navigate to the `Cleaned Data SET/` folder to review the cleaned data.
2. Download and open `MarketingAnalysis.xlsx`.
3. Interact with the slicers at the top of each page to drill down into the insights.
